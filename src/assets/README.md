# Assets Folder

This folder is for storing images, icons, and other static assets.

## To use images:
1. Add your images here (e.g., hero.jpg, car.png, etc.)
2. Reference them in your JSX as:
   - Absolute path: `/src/assets/image.jpg` 
   - Import: `import img from './assets/image.jpg'`
   - Then use: `<img src={img} />`

## Example images for this project:
- `hero.jpg` - Hero section background (1920x1080px recommended)
- `car-service.jpg` - Service showcase image
- `team-member.jpg` - Team member photos
